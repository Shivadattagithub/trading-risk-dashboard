# 📊 TradeRisk Dashboard

> A free, open-source trading journal and risk management tool — built for serious traders.

**Live Demo:** [Click here to try it](https://shivabotla.github.io/trading-risk-dashboard)

Built by [Shiva Botla](https://github.com/ShivaBotla) — Quant & Discretionary Trader with experience in FX, derivatives, and crypto markets.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📓 **Trade Journal** | Log every trade with entry, exit, SL, size, and notes |
| 📊 **Performance Dashboard** | Win rate, P&L curve, profit factor, Sharpe ratio, max drawdown |
| 🛡️ **Risk Calculator** | Position sizer + R:R finder + drawdown simulator |
| 🌐 **Macro Event Tracker** | Plan trades around CPI, NFP, FOMC and other catalysts |

## 🚀 How to Use

**Option 1 — Just open it (no install needed):**
1. Download or clone this repo
2. Open `index.html` in any browser
3. Start logging trades — everything saves locally in your browser

**Option 2 — Use the live version:**
Visit the GitHub Pages link above.

## 📁 File Structure

```
trading-risk-dashboard/
├── index.html    ← Main app (open this in your browser)
├── style.css     ← All styling
├── app.js        ← All logic (journal, charts, risk calc)
└── README.md     ← This file
```

## 🧠 Built With

- Vanilla HTML, CSS, JavaScript (zero frameworks, zero dependencies to install)
- [Chart.js](https://www.chartjs.org/) for charts (loaded from CDN)
- Google Fonts (Space Mono + Syne)
- LocalStorage for data persistence

## 📈 Who Is This For?

- Retail traders who want to track performance seriously
- Quant traders building discipline around systematic journaling
- Anyone trading FX, crypto, or derivatives who wants a free alternative to expensive journal apps
